# 前端项目深度分析

当用户调用此 Skill 时，读取 `prompts/analysis.md`，将其完整内容作为 prompt 传入 Agent 工具（`subagent_type: general-purpose`），启动分析代理执行前端深度分析。

分析代理返回的报告**必须完整呈现**给用户，不得省略、压缩或改写任何章节。即使报告很长，用户也期望看到完整的九章分析内容。如果返回内容被截断，分段请求剩余部分。
